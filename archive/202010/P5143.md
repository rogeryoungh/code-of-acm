# P5143 攀爬者

## 题目大意

在三维空间中，HKE 只能往上走，求攀爬总长。

## 分析

对所有座标的 $z$ 分量排序即可。

```cpp
struct POINT {
    double x, y, z;
    bool operator<(POINT p) {
        return z > p.z;
    }
} nn[50010];

int main() {
    int n;
    scanf("%d", &n);
    _fora (i, 1, n)
        scanf("%lf%lf%lf", &(nn[i].x), &(nn[i].y), &(nn[i].z));
    sort(nn + 1, nn + n + 1);
    double len = 0;
    _fora (i, 2, n) {
        double t = (nn[i].x - nn[i - 1].x) * (nn[i].x - nn[i - 1].x);
        t += (nn[i].y - nn[i - 1].y) * (nn[i].y - nn[i - 1].y);
        t += (nn[i].z - nn[i - 1].z) * (nn[i].z - nn[i - 1].z);
        len += sqrt(t);
    }
    printf("%.3lf\n", len);
    return 0;
}
```
