## 题目大意

给出正整数 $n$ 和 $k$，请计算

{{< display-math >}}
G(n, k) = \sum_{i = 1}^n k \bmod i
{{< /display-math >}}

## 分析

因为

{{< display-math >}}
k \bmod i = k - i \left\lfloor \frac{k}{i} \right\rfloor
{{< /display-math >}}

因此有

{{< display-math >}}
G(n,k) = nk - \sum_{i=1}^n i \left\lfloor \frac{k}{i} \right\rfloor
{{< /display-math >}}

后面需要整数分块。

```cpp
int main() {
    ll n = rr();
    ll k = rr();
    ll sum = n * k;
    n = min(n, k);
    for (ll l = 1, r; l <= n; l = r + 1) {
        r = min(n, k / (k / l));
        ll t = (l + r) * (r - l + 1) / 2;
        sum -= t * (k / l);
    }
    printf("%lld", sum);
    return 0;
}
```
