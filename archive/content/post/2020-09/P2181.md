## 题目大意

凸 $n$ 边形中，任意三条对角线不共点，求所有对角线交点的个数。

## 分析

注意到一个交点对应凸多边形 $4$ 个定点，于是等价于 $n$ 个点任选 $4$ 个点的选法种数，即

{{<display-math>}}
\binom{n}{4} = \frac{n(n-1)(n-2)(n-3)}{24}
{{</display-math>}}

注意爆 `long long`，需要写成 `n * (n-1) / 2 * (n-2) / 3 * (n-3) / 4`。

```cpp
int main() {
    unsigned long long n;
    scanf("%llu", &n);
    n = n * (n - 1) / 2 * (n - 2) / 3 * (n - 3) / 4;
    printf("%llu", n);
    return 0;
}
```
